# Tech Stack - Dice of Destiny (Ver2/Ver3)

## Frontend
- **HTML5:** Semantic markup for all application components (Game, Surveys, Landing Page).
- **CSS3:** Modern CSS utilizing Custom Properties (Variables) for themed, consistent styling and Flexbox/Grid for responsive layouts.
- **Vanilla JavaScript:** Framework-less JavaScript for DOM manipulation, game logic, and survey handling.

## Backend & Integration
- **Google Apps Script (GAS):** Serves as the middleware to receive JSON data from the frontend and interact with Google Sheets.
- **Google Sheets API:** Used via GAS for persistent data storage and organization.

## External Assets & Services
- **Google Fonts:** Integration of "Cinzel" and "Lora" for consistent typography.

## Development & Deployment
- **Static Hosting:** The application is composed of static assets (HTML, CSS, JS) suitable for hosting on services like GitHub Pages.
- **Version Control:** Git for source code management.
